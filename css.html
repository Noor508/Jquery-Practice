  1. Tag based Selector: $("TAG_Name")
  2. Id-based selector: $("#id")
  3. class-based selector: $(".className")
  4. Mixed: $("p#para") paragraph tag having id= para
  5. class and tag based: $("div.dummy") div having class dummy
  6. space means inside
    (tag.className paragraph): $(div.dummy p) : div with dummy class and all p tags in it.
  7. comma separator: it means or $(tag1 ,tag2)
